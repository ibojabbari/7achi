7achi (حچي) (meaning “conversation” in Iraqi Arabic) is a simple social media–style web application built with Django

Despite its name, the project does not include a real chat system. Rather, it simply focuses on simulating the behavior of a social media website where users can share in a shared feed

The main purpose of this project is to demonstrate my understanding of how Django works internally, in a very compact and basic program in order to avoid complexity, as it was made for educational purposes.
